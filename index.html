Firebase Configuration
⚠️
Firebase belum dikonfigurasi. Ikuti langkah di bawah.
ℹ️
Cara setup Firebase (gratis, 5 menit):
1. Buka console.firebase.google.com → login Google
2. Klik "Create a project" → nama: qufarma → buat
3. Klik ikon </> (Web) → daftarkan app → salin firebaseConfig
4. Di sidebar → Build → Firestore Database → Create → pilih test mode
5. Di sidebar → Build → Storage → Get started → test mode
6. Isi kolom di bawah dengan nilai dari firebaseConfig
🔗
Bagikan Konfigurasi ke Device Lain

Generate link atau QR code berisi semua konfigurasi Firebase. Buka di device lain agar langsung tersinkron tanpa perlu input ulang.
